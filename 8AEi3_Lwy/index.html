第十届“政治文化、心理与行为研究”工作坊 征文启事

发布于 2022-04-18 | 4分钟 | 1018字数 | 浏览量：

“政治文化、心理与行为研究”工作坊自2013年创办以来，至今已成功举办九届，得到了学界同人的大力支持，取得了良好的学术反响。为进一步交流和研讨学界有关政治文化、心理与政治行为领域研究的最新成果，推动政治科学研究在中国的发展和进步，2022年七八月间南京大学公共事务与地方治理研究中心、南京大学政府管理学院将于南京市主办第十届“政治文化、心理与行为研究”工作坊，特向境内外同人诚邀优秀成果并莅会。现将相关事宜预告如下。

一、征文选题范围

举凡应用社会科学方法，研究政治文化、心理、行为与社会政治问题的科学论文，均在征集之列，如下主题（包括但不限于）的实证研究成果，尤受欢迎。

  1. 中国政治与政治科学的基本概念：建构与测量；
  2. 政治科学研究方法的转型、发展及其反思；
  3. 政治心理学的前沿研究；
  4. 政治文化的比较研究；
  5. 政治参与、政治行为的前沿研究；
  6. 地方与国家治理的前沿研究。

二、征文要求

  1. 论文应为未公开发表过的原创文章，字数一般不超过2万字。
  2. 文章的内容与格式，应按照最新的《政治学研究》杂志的文稿体例与引注规范进行编排。论文首页脚注处须注明“作者简介”，包括姓名、工作单位、职称/职务、通信地址、电子邮件、主要研究方向等内容。
  3. 鼓励学界同人作为组稿人，围绕本届工作坊征文选题进行组稿和邀稿，鼓励跨单位、跨学科相互交流，每组稿件以3篇左右为宜，请按要求将所组专题的稿件与内容简介汇总提交。
  4. 请在2022年6月25日前将征文回执（“回执”请见附件）与论文全文（命名为“单位+姓名+论文题目”），以word格式发送至工作坊投稿邮箱shwdnju@163.com。个人投稿，邮件主题请标注为：“单位+姓名+第十届工作坊”；专题组稿，邮件主题请标注为：“组稿+专题名称+第十届工作坊”。
  5. 本届工作坊将邀请专家对提交的论文进行评审，并遴选部分高质量论文向国内相关学术期刊推荐发表。
  6. 评审结果将于2022年7月1日前公布，组委会将通过电子邮件向入选作者发送工作坊邀请函。入选名单与会务信息将发布在南京大学公共事务与地方治理研究中心的微信公众号“NJU公共事务与地方治理”（shwdnju）与官网（shwd.nju.edu.cn），以及南京大学政府管理学院官网（public.nju.edu.cn）等平台，敬请关注。

三、参会须知

  1. 会议时间：2022年七八月间（具体时间待定）。
  2. 会议地点：南京大学仙林校区国际会议中心。
  3. 会议形式、时间及地点可能根据疫情形势及防控政策做出相应调整，工作坊具体安排将另行通知。
  4. 会议费用：本届工作坊不收取会务费，会议期间与会者的食宿费用将由组委会提供，往返交通费用请自理。
  5. 会议组联系方式：
    （1）刘文超，南京大学公共事务与地方治理研究中心博士研究生，电话：18522760512。
    （2）郭军琦，南京大学公共事务与地方治理研究中心博士研究生，电话：18375860147。
    （3）会务邮箱：shwdnju@163.com。
